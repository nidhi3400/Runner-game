# Endless Runner Game

A browser-based endless runner built with PIXI.js and TypeScript.

## Features

- 3-lane runner mechanics
- Smooth lane switching
- Infinite scrolling road animation
- Random obstacle spawning
- Collision detection
- Game over state
- Responsive canvas rendering

## Tech Stack

- TypeScript
- PIXI.js
- Vite

## Roadmap

- Score system
- Difficulty scaling
- Coin collection
- Sprite assets
- Spine animations
- Mobile swipe controls
